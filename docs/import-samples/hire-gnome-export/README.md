# Hire Gnome Export Sample

This folder contains a sample `Hire Gnome Export` ZIP for `Admin Area > Data Import`.

Contents:
- `manifest.json`
- `metadata.json`
- `data/customFieldDefinitions.json`
- `data/clients.json`
- `data/contacts.json`
- `data/candidates.json`
- `data/jobOrders.json`
- `data/submissions.json`
- `data/interviews.json`
- `data/placements.json`

Use this sample to test the native Hire Gnome export import path.
